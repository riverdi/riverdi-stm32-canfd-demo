void setL8Palette ( const uint8_t *const data)

Sets L8 palette to be used by the painter. The content pointed to be parameter data depends on the actual L8 format. L8RGB8888 assumes data points to three bytes per palette index,. L8ARGB8888 assumes four bytes per palette index.
