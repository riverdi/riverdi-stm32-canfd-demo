Detailed Description

The PainterRGB888 class allows a shape to be filled with a given color value. This allows anti-aliased elements to be drawn.

◆ paint()

void paint ( uint8_t * destination,
int16_t offset,
int16_t widgetX,
int16_t widgetY,
int16_t count,
uint8_t alpha ) const
virtual

Paints a streak of pixels (all with the same alpha) in the framebuffer. The first pixel to paint is at 'offset' relative to 'destination' (to be able to support 1bpp, 2bpp and 4bpp). The first pixel to paint is at the given 'widgetX', 'widgetY' coordinate. The number of pixels to paint is 'count' and the alpha to apply is 'alpha'.

Note that the widgetX, widgetY is relative to the widget in the framebuffer, not the display. To convert the coordinate to display widget coordinates, use framebufferToDisplay().

Parameters
[in]destinationIf non-null, the pointer.
offsetThe offset to add to the destination.
widgetXThe widget x coordinate.
widgetYThe widget y coordinate.
countNumber of pixels.
alphaThe alpha of the pixels.
